# FastAPI + React Tutorial — Budget Tracker

A hands-on, lab-based tutorial for building a full-stack web app with FastAPI (backend) and React (frontend). You'll build a real personal budget tracker from scratch.

## Who This Is For

You have basic programming experience (Python or JavaScript) but haven't built a web app before. No prior FastAPI or React knowledge required.

## What You'll Build

A personal budget tracker that lets you:
- Record income and expense transactions
- Categorize and filter transactions
- View your balance and spending summary
- Set monthly budget limits per category

## Prerequisites

- Python 3.11+
- Node.js 18+
- A terminal
- A code editor (VS Code recommended)

## How to Use This Tutorial

Work through the labs in order. Each lab has three files:
- `README.md` — concepts and guided walkthrough
- `SPEC.md` — what you need to implement
- `EXPECTED.md` — how to verify your work

Do not skip labs — each one builds on the previous.
